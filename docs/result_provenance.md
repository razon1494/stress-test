## Current result set

- Dataset: 5,294 records representing 3,932 document identifiers.
- Core machine-text source: HC3.
- Detectors: seven.
- Evaluation conditions: fourteen total—clean plus thirteen transformed
  conditions present in the score cache.
- Primary README table: `results/per_domain.json`.
- Manuscript status: working draft, not a released preprint.

## Interpretation rules

1. Label the numbers **preliminary** until independent threshold calibration and
   human semantic-preservation validation are complete.
2. Do not combine pooled-threshold and per-domain results in one claim without
   naming both regimes.
3. Do not treat subgroup differences as causal evidence about writer identity.
   Register, prompt, proficiency, sample size, and detector mechanism are
   entangled in the current observational slices.
4. Include sample sizes and uncertainty for subgroup claims.
5. When results are regenerated, update the structured JSON first, then the
   Markdown summaries, README, manuscript, and figure captions in that order.

## Files intentionally not committed

- Licensed raw corpora under `data/external/`
- Processed/transformed text
- Per-example score caches
- Fine-tuned model weights
- In-progress annotation responses

These exclusions are required for licensing, privacy, and repository-size
reasons. A reproducible release should publish permitted artifacts separately
with checksums and versioned metadata.
